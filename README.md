# 🌌 Солнечная система

Telegram Mini App с интерактивной визуализацией Солнечной системы в реальном времени.

## Возможности

- Реальные положения всех 8 планет и Плутона на текущую дату
- Луны: Луна, Фобос и Деймос, галилеевы спутники Юпитера, Титан, Энцелад, Тритон, Харон и другие
- Кольца Сатурна
- Симуляция времени (1 секунда = 1 сутки), кнопка возврата к текущей дате
- Тап по планете — подробная информация: расстояния, период, диаметр, скорость, описание
- Zoom-вид по двойному тапу: планета крупным планом со спутниками на орбитах
- Тап по спутнику — информация о спутнике с фото
- Кометы и метеоры в фоне

## Стек

- Чистый HTML + CSS + JS, без фреймворков
- Canvas 2D
- [Astronomy Engine](https://github.com/cosinekitty/astronomy) (Don Cross, MIT) — точные астрономические расчёты
- Telegram WebApp SDK

## Данные и изображения

Астрономические расчёты основаны на алгоритмах [JPL (NASA)](https://ssd.jpl.nasa.gov/), реализованных в библиотеке Astronomy Engine.

Фотографии планет и спутников: NASA / ESA / JPL / Wikimedia Commons (Public Domain).
